## Synopsis

React component for easy build rich-editors. 
Package also provide a few preconfigured components.

## Code Example

```
import { RichEditor } from 'jcatalog-react-rich-editors/lib/RichEditor';
...
<RichEditor />
```

* Run `npm install`
